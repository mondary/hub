Avec la touche ALT il est possible de déplacer la ligne de haut en bas
Pour commenter, il suffit de surligner le texte et faire CMD + SHIFT + /
Ici on va charger le fichier index4.js
Pour formater le code automatiquement tu peux faire :
- clic droit > format with > PRETTIER
- aller dans les settings > Defaut Formater > choisir PRETTIER, puis en dessous cocher Format on save (a chaque fois que je sauvegarde, le code sera corrigé, indenté, etc ...)
